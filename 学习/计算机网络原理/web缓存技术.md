# 功能

在不访问服务器的前提下满足客户端的网络请求

- 缩短客户请求的响应时间
- 减少组织/机构的流量
- 实现大范围数据的分发

# Web 缓存服务器

用户访问 Web 缓存服务器，缓存服务器对用户进行响应
缓存服务器访问服务器，并对服务器信息进行缓存

一般由 ISP 服务商架设缓存服务器

# 好处
- 可以迅速的响应某些请求
- 降低了平均延迟

# 坏处
- 已缓存的链接的时效性（得到了解决）
- 对于未缓存的提升了延迟（访问延迟）
